# 🎬 Netflix Movies & TV Shows - Excel Dashboard Project

This project is an Excel-based interactive dashboard built using the **Netflix Movies & TV Shows dataset** (sourced from Kaggle). It demonstrates key skills in data analysis, data cleaning, and visualization using advanced Excel features.

---

## 📌 Project Objectives

- Analyze global Netflix content trends
- Identify most-watched genres
- Discover top contributing countries and years
- Display content type breakdown (TV Shows vs Movies)
- Present key metrics using KPI Cards and Charts

---

## 📂 Dataset Source

**Netflix Movies and TV Shows Dataset**  
📥 From: [Kaggle - Netflix Titles](https://www.kaggle.com/datasets/shivamb/netflix-shows)  
Fields used:
- Title
- Director
- Cast
- Country
- Date Added
- Duration
- Listed_in (Genres)
- Type (Movie or TV Show)

---

## 📊 Excel Skills Demonstrated

- ✅ **Data Cleaning** (removing blanks, formatting dates, standardizing text)
- ✅ **Functions**: `IF`, `IFERROR`, `VLOOKUP`, `INDEX`, `MATCH`, `OFFSET`, `AND`, `OR`, `SUMIF`, `COUNTIF`, `TEXT`, `DATEDIF`, etc.
- ✅ **Pivot Tables**: Genre-wise viewers, Country content share, Year-wise trends
- ✅ **Pivot Charts**: Column, Pie, Bar, Line, and Slicers
- ✅ **KPI Cards** using formulas and conditional formatting
- ✅ **Slicers** for interactive filtering
- ✅ **Power Query** (optional stage for genre splitting and cleanup)
